# Online  Learning  System 

A role-based Learning Management System built with Laravel.
This system supports **Admin, Teacher, and Student dashboards** with course management, assignments, and enrollment features.



***  Features

#  Authentication

* Custom login system (role-based)
* Admin / Teacher / Student access control
* Secure logout system

# Admin Panel

* Manage users (Admin, Teacher, Student)
* View total users, teachers, students
* Create & delete users
* Course management

#  Teacher Panel

* Create courses
* Add assignments
* Add live classes
* Upload recorded classes
* View student submissions
* Grade assignments with feedback

#  Student Panel

* Enroll in courses
* View enrolled courses
* Submit assignments
* See marks & feedback
* Track deadline (early / late submission)


#  Technologies Used

* Laravel Framework
* Blade Template Engine
* MySQL Database
* Bootstrap
